FReport v1.01

Генератор отчетов.

Оболочка для широко известного генератора отчетов FastReport v2.46, http://www.fastreport.ru

-Шаблоны отчетов хранятся в файлах с расширением *.frf. 
-Файл шаблона выбирается упрощенным навигатором в файловой системе.
-Шаблоны можно открывать в режиме дизайна, либо сразу в режиме просмотра результата отчета.
-В файлах шаблонов полностью сохраняется структура отчета fastreport: дизайн, формы диалогов запроса параметров для отчета, SQL-запросы к базам, переменные отчета. 
-Соединения с базами (Database), используемыми на вашем предприятии, можно создать списком и сохранить в реестре, либо в файлах cfg.
-Доступ к базам через BDE. 
-Прямой доступ к Interbase (Interbase Express 6.02).
-Прямой доступ к Oracle (библиотека NCOCI8, http://www.da-soft.com )
-Экспорт в форматы rtf, txt, csv, htm.
-Поддержка RTF 2.0, CheckBox, диаграмма, штрих-код, ole-объект, кросс-таб, подотчет.
-Несколько демонстрационных отчетов.
-Руководство пользователя fastreport на русском языке.
-Интерфейс полностью на русском языке.
