FReport v2.3

Универсальный генератор отчетов

Оболочка для набора компонентов FastReport v2.54, http://www.fastreport.ru

-Шаблоны отчетов хранятся в файлах с расширением *.frf. 
-Файл шаблона выбирается упрощенным навигатором. Путь к папке с отчетами автоматически запоминается.
-Шаблоны можно открывать в режиме дизайна-редактирования, либо сразу запускать отчет с предварительным отображением диалога ввода параметров.
-Возможна установка приложения непосредственно пользователю.
-В файлах шаблонов полностью сохраняется структура отчета fastreport: дизайн, формы диалогов запроса параметров для отчета, SQL-запросы к базам, переменные отчета. 
-Пул именованых соединений с базами данных, используемыми в организации, можно создать один раз и сохранить в реестре, либо в файлах cfg. Пул может загружатся автоматически при запуске приложения, любой sql-запрос внутри отчета может использовать любое из загруженых соединений.
-Доступны подключения к базам:
--источник данных ODBC
--источник данных ADO (возможны выборки из MSSQL, Excel)
--OCI подключения к Oracle 
--Подключения к Interbase/Firebird
-Одновременный сбор и соединение данных в одном отчете из различных источников.
-Экспорт в форматы PDF, JPG, RTF, CSV, XLS, HTML, TXT, XML. Печать на любой принтер в системе.
-Отображение в отчете rtf, checkbox, image, простая диаграмма, штрих-код, ole-объект, кросс отчет (горизонтальный), отчеты с различными формами страниц.
-Руководство пользователя fastreport на русском языке.
-Интерфейс полностью на русском языке.
-Написано на Delphi 7
